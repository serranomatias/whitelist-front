# Whitelist contract Dapp

you can access on: whitelist-front.vercel.app

The smartcontract allow to x quantity of users be whitelisted.
If limit is reached don't allow to add more users.

The front-end use: 
- Next.js
- dotenv
- web3modal
- ethers.js
